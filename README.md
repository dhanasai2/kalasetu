## 🔥 The Problem

| Reality | Impact |
|---------|--------|
| **90%** of Indian artisans have zero digital presence | Invisible to 500M+ online shoppers |
| No photography, writing, or marketing skills | Can't create product listings |
| Middlemen capture **40–60%** margins | Artisans earn < ₹200/day |
| **50%** of traditional craft forms lost in 50 years | Cultural heritage disappearing |
| Language barriers across **22+** languages | Locked out of e-commerce entirely |

## ✨ KalaSetu's Solution

> **कलासेतु** (Kala = Art, Setu = Bridge) — An AI-powered marketplace that transforms a single craft photo into a complete digital storefront in 60 seconds, in the artisan's own language.

**No typing. No English. No digital skills needed.**

## ⚡ Quick Start

### Prerequisites

- Python 3.10+
- pip

### 1. Clone

```bash
git clone https://github.com/dhanasai2/kalasetu.git
cd kalasetu
```

### 2. Virtual Environment

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS / Linux
source .venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
pip install requests reportlab qrcode[pil]
```

### 4. Configure Environment

```bash
# Create .env in project root
```

```env
GROQ_API_KEY=your_groq_api_key_here
DJANGO_SECRET_KEY=your-secret-key-here
DEBUG=True
```

> 🔑 Get a free API key at [console.groq.com](https://console.groq.com)

### 5. Database Setup

```bash
python manage.py migrate
python manage.py seed_data    # Optional: loads sample artisans & products
```

### 6. Run

```bash
python manage.py runserver
```

🎉 Open **http://127.0.0.1:8000** — KalaSetu is live!
